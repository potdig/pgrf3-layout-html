# Layout HTML Builder for PGRF3

## Build
```
npm run build
```

## Test
```
docker compose build
docker compose up [-d]
```
or
```
npm run dev
```

## Tips
* 下記のカスタム CSS を OBS で設定することで映像配置エリアを完全透明にできます
  ```css
  .video {
    background-color: rgba(0, 0, 0, 0) !important;
  }
  ```
